# Bilingual-Sentiment-Analysis
Mini project: Analyzing text sentiment in Arabic and English via GitHub Models

## Objectives
- Build a bilingual sentiment analysis tool using GPT-4o-mini.
- Learn how to connect GitHub Marketplace models to Python via API.
- Practice prompt design and user input handling in Colab.

## Steps Completed
1. Installed OpenAI client and set up API access.
2. Created sentiment analysis function for English + Arabic.
3. Tested model with sample inputs.
4. Prepared notebook for submission.
